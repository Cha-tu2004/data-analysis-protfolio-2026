
# 蓝莓产量预测分析

## 项目背景
基于15,289条蓝莓农业观测数据，探究环境因素对产量的影响，并构建预测模型。

## 技术栈
- Python 3.x
- pandas, numpy（数据处理）
- matplotlib, seaborn（可视化）
- scikit-learn（机器学习）
- statsmodels（统计检验）

## 核心分析流程
1. **数据探查**：处理15K+数据，无缺失值、无重复值
2. **探索性分析**：相关性热力图发现温度指标高度共线性
3. **聚类分析**：K-Means分4类，识别高产/低产群体特征
4. **特征工程**：VIF检测发现严重共线性，采用PCA降维保留95%信息
5. **建模预测**：线性回归，R² = 0.787
6. **模型诊断**：残差分析验证模型假设成立

## 关键结论
- 蜜蜂授粉密度是产量的核心驱动因素
- 连续阴雨天显著抑制产量
- PCA有效消除共线性，提升模型稳定性
