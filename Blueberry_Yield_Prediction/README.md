
# 蓝莓产量预测分析

## 🌿 项目背景
基于 15,289 条蓝莓农业观测数据，探究环境因素（气候、授粉昆虫等）对产量的影响，并构建预测模型帮助农场主预估收成。

## 📈 数据集
- ​**​训练集​**​：15,289 行 × 18 列
- ​**​测试集​**​：10,194 行 × 17 列
- ​**​数据质量​**​：无缺失值、无重复值，数据完整性良好

## 🔍 分析流程

### 1. 数据探索（EDA）
- 绘制相关性热力图，发现温度指标间存在高度共线性
- 识别出核心驱动因素：果实数量（fruitset）、种子数（seeds）

### 2. 聚类分析（K-Means）
- 使用肘部法则与轮廓系数确定最佳聚类数 ​**​K=4​**​
- 成功识别出高产组、中产组、低产组及特殊气候组

### 3. 特征工程
- ​**​VIF检测​**​：发现严重多重共线性（部分 VIF > 30,000）
- ​**​PCA降维​**​：将 17 个特征压缩为 7 个主成分，保留了 ​**​95%​**​ 的信息量，有效解决了共线性问题

### 4. 建模预测
- 采用线性回归模型进行训练
- ​**​测试集 R² = 0.787​**​，MSE = 393,493

### 5. 模型诊断
- 残差时序图：无明显自相关
- 残差直方图 & Q-Q图：符合正态分布假设
- 残差-预测值图：无明显异方差，模型稳健

## 💡 核心结论
1. ​**​生物学特性主导​**​：果实数量和种子数是产量的最强预测因子。
2. ​**​授粉重要性​**​：蜜蜂密度对高产群体有显著正向影响。
3. ​**​气候风险​**​：连续阴雨天是抑制产量的重要风险因素。
