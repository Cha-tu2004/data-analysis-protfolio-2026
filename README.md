# 电商销售数据分析

## 分析目标
找出销售额增长的机会点和风险点，优化广告投放策略。

## 关键发现
1. **渠道策略分化**：小程序"赌爆款"（ROI最高但波动大），APP"烧钱换量"（订单多但效率低）
2. **地区价值被低估**：西北地区ROI最高，应加大投放
3. **用户价值差异**：老客ROI是新客的2倍以上，维护老客比拉新划算

## 技术栈
- Python：pandas, numpy, matplotlib, seaborn
- Power BI：DAX度量值、散点图、条形图、柱状图
